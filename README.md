# Computer_Vision_Midterm

Project name: Face Mask Detector 

Team members: Evan Gibson

Tier 1 (First CV project)

Problem: People often do not like wearing masks or wear them incorrectly. This can cause many health issues in medical environments.

Solution Overview: This model will take an image of them and determine whether their mask is on and worn correctly. This can allow them to enter or access an area that requires them to do so without the danger they could pose without a mask.

Technical Approach: Faster R-CNN, PyTorch, Colab

Dataset Plan: Kaggle, ~1000 size, labels(no_mask, mask_wrong, mask_right) https://www.kaggle.com/datasets/vijaykumar1799/face-mask-detection

Metrics: Accuracy and Precision


Week by Week
10:	Get dataset, set up environment  
11: Train or fine-tune model	
12: Test and improve	
13: Create demo or video	
14: Final testing, documentation	

Plan B's:
Not enough data -> Switch to ImageNet set    
Low accuracy -> Fine tune, more data
